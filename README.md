# 长江对话，世界回响 (Changjiang Dialogue, World Echo)

一个现代化的网站应用，展示长江对话平台的信息，并包含两个经典游戏：国际象棋和俄罗斯方块。

## 项目结构

```
changjiang-dialogue-web/
├── assets/
│   └── images/          # 图片资源文件夹
├── scripts/
│   ├── main.js          # 主页面JavaScript脚本
│   ├── chess.js         # 国际象棋游戏逻辑
│   └── tetris.js        # 俄罗斯方块游戏逻辑
├── styles/
│   ├── style.css        # 主页面样式表
│   ├── chess.css        # 国际象棋游戏样式
│   └── tetris.css       # 俄罗斯方块游戏样式
├── index.html           # 主页面
├── login.html           # 登录页面
├── news.html            # 新闻页面
├── chess.html           # 国际象棋游戏页面
├── tetris.html          # 俄罗斯方块游戏页面
└── .gitignore           # Git忽略文件
```

## 功能特性

### 主页面 (index.html)
- 响应式导航栏，支持滚动隐藏/显示
- 英雄区域展示平台主题
- 功能特点卡片展示
- 兼容性说明
- 下载按钮及模拟下载功能
- 常见问题(FAQ)展开/折叠功能
- 语言选择器
- 平滑滚动效果
- 滚动动画

### 登录页面 (login.html)
- 用户登录表单
- 简单的表单验证
- 导航栏和页脚

### 新闻页面 (news.html)
- 新闻列表展示
- 分页控件
- 导航栏和页脚

### 国际象棋游戏 (chess.html)
- 完整的国际象棋规则实现
- 3D立体视觉效果
- 实时回合指示器
- 悔棋功能
- 新游戏功能
- 积分系统（胜利方+10分，失败方+1分）
- 玩家名称输入
- 本地存储排行榜（显示前10名）
- 游戏结束判定

### 俄罗斯方块游戏 (tetris.html)
- 完整的俄罗斯方块规则实现
- 星际主题视觉效果
- 分数、等级、行数显示
- 下一个方块预览
- 键盘控制（方向键移动，空格快速下落）
- 游戏暂停/继续功能
- 本地存储排行榜（显示前5名）
- 游戏结束模态框

## 技术栈

- **HTML5**：页面结构和语义化标签
- **CSS3**：样式设计，包括：
  - 响应式设计
  - 3D变换和动画
  - 渐变效果
  - Flexbox和Grid布局
- **JavaScript (ES6+)**：交互逻辑和游戏实现
- **LocalStorage**：游戏数据本地存储

## 安装说明

1. 克隆或下载项目到本地：
   ```bash
   git clone <repository-url>
   cd changjiang-dialogue-web
   ```

2. 启动本地服务器（推荐使用Python的HTTP服务器）：
   ```bash
   python -m http.server 8000
   ```

3. 在浏览器中访问：
   ```
   http://localhost:8000
   ```

## 使用方法

### 主页面
- 浏览平台信息
- 点击导航链接跳转到不同区域
- 使用下载按钮模拟下载
- 点击FAQ问题展开查看答案
- 使用语言选择器切换语言（模拟）

### 登录页面
- 输入用户名和密码
- 点击登录按钮提交表单

### 新闻页面
- 浏览新闻列表
- 使用分页控件查看更多新闻

### 国际象棋游戏
1. 在玩家名称输入框中输入白方和黑方玩家名称
2. 点击棋盘上的棋子选择
3. 点击目标格子移动棋子（高亮格子表示有效移动）
4. 点击"悔棋"按钮撤销上一步移动
5. 点击"新游戏"按钮开始新游戏
6. 游戏结束后，点击"结束游戏"按钮，输入获胜方，系统会自动更新积分和排行榜

### 俄罗斯方块游戏
1. 点击"开始游戏"按钮开始游戏
2. 使用方向键控制方块移动：
   - 左箭头：向左移动
   - 右箭头：向右移动
   - 下箭头：加速下落
   - 上箭头：旋转方块
   - 空格：快速下落到底部
3. 点击"暂停"按钮暂停/继续游戏
4. 点击"重置"按钮重新开始游戏
5. 游戏结束后，输入玩家名称保存分数到排行榜
6. 点击"排行榜"按钮查看历史最高分
